## 2024-04-19 - Adding clear button to search input
**Learning:** Adding a clear search button inside search inputs significantly improves UX by saving users from repeatedly hitting backspace. Proper padding is required (`pr-10`) so the text doesn't overlap the clear icon. Accessibility labels are crucial for clear icons and filtering triggers.
**Action:** Always include a "clear" `X` icon inside search inputs when they are populated with text, ensuring adequate padding and a keyboard focus state.
