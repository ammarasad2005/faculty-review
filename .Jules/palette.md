## 2024-05-24 - Input Clear Button Pattern\n**Learning:** Users need a quick way to reset search filters, especially on mobile where deleting text character-by-character is tedious. A clear button improves navigation speed.\n**Action:** For all future text inputs used for searching or filtering, include a conditionally rendered "Clear" (X) button on the right side. Ensure it uses `type="button"`, has an `aria-label`, and includes `focus-visible` ring styling for keyboard users.
